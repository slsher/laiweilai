1.算法题
你截获了一份敌国词典，其语言使用的字母和英文相同，但顺序不同。
这部辞典中单词的顺序按照其语文字母顺序排列, 需要你分析出其国家语言的字母排序。
例子，若字典顺序如下：["xmc","xmq","em","ecc", "mqcc"]，正确顺序是"xemcq"
